# Sign Language App

## 📌 Descripción

Aplicación desarrollada para interpretar lenguaje de señas.

## 🛠️ Tecnologías utilizadas

* JavaScript
* React
* Node.js

## 📁 Estructura del proyecto

* src/
* public/
* package.json

## ▶️ Cómo ejecutar

1. Clonar repositorio:
   git clone https://github.com/TU_USUARIO/senas-app.git

2. Instalar dependencias:
   npm install

3. Ejecutar:
   npm start
